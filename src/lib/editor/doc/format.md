卡片類別

1. [[TOPIC]]：針對一個`主題`的卡片，包括

   - [[主題名]]：主題、概念、物
   - [[事件名]]：事件
   - [[Q:......]]：問題，用

2. $TICKER：例如$BA、$TSLA，針對一個金融商品的卡，本質上仍是[[主題]]卡，但可以用`$`連結
3. Web Card：針對一個網頁的筆記卡
