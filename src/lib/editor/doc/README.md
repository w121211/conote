符號定義
[[主題標題]]：可自定義、用[[ＯＯＯ]]包著
$BA：股票、金融商品代號，以`$`為頭
[~]：關鍵字，需為 [[Topic]] 或 $Ticker，用空格隔開
[*]：重點、摘要、TLDR
[?]：問題
[+]：優勢、正面
[-]：劣勢、負面
[自定義]：可自定義任意標題
%OOO：該行的識別 ID，不可刪除
\-\-\-：
===：新卡片的分界線
